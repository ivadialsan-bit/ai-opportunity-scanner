# Demo 48H page

Демо-страница для ручной продажи оффера “Цифровая точка заявки за 48 часов”.

Файл:
- `public/demo-48h/index.html`

Назначение:
- показать клиенту пример результата;
- не использовать бренд клиента без согласия;
- не обещать рост заявок;
- продавать порядок в приёме обращений.

Локальный запуск:

`cd /root/ai-opportunity-scanner/public && python3 -m http.server 8080 --bind 0.0.0.0`
