# LEAD_SCORING_SPEC

## Цель

Lead Score должен показывать, кому писать первым.

## Шкала

0–100.

## Баллы

- Ниша горячая/сезонная: +15
- Есть телефон: +10
- Есть 10+ отзывов: +15
- Рейтинг 3.7–4.6: +10
- Нет сайта: +15
- Есть сайт, но нет формы заявки: +10
- Нет онлайн-записи: +10
- Нет мессенджер-ссылки: +5
- Слабое описание услуг: +10
- Работает сейчас: +5
- Есть фото/активная карточка: +5

## Интерпретация

- 70+ — писать сегодня.
- 50–69 — вторая очередь.
- ниже 50 — не тратить время на первой неделе.

## Важное правило

“Нет сайта” само по себе не делает лида хорошим.

Хороший лид:
- есть спрос;
- есть отзывы;
- есть телефон;
- есть коммерческая услуга;
- есть видимая цифровая дыра.

Плохой лид:
- нет отзывов;
- нет бюджета;
- нет признаков активности;
- одиночный мастер без системы;
- крупная сеть с долгим циклом согласования.
